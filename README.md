toriaezu
========

Install some tools and dotfiles.

How to use
----------

```console
git clone https://github.com/shakiyam/toriaezu
cd toriaezu
./provision.sh
```

Target OS
---------

* Amazon Linux AMI 2017.03
* Oracle Linux Server 7.4 with UEK4
* Oracle Linux Server 6.9 with UEK4 (experimental)
* Ubuntu 16.04.1 LTS

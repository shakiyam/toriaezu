toriaezu
========

Install some tools and dotfiles.

## How to use ##

```
git clone https://github.com/shakiyam/toriaezu
cd toriaezu
sudo ./provision.sh
```

## Target OS ##

* Amazon Linux AMI 2016.09
* Oracle Linux Server 7.3 with UEK4
* Oracle Linux Server 6.8 with UEK4 (experimental)
* Ubuntu 16.04.1 LTS

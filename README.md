toriaezu
========

Install some tools and dotfiles.

How to use
----------

```console
git clone https://github.com/shakiyam/toriaezu
cd toriaezu
./provision.sh
```

Target OS
---------

* Oracle Linux Server 7.7 with UEK5
* Ubuntu 18.04 LTS

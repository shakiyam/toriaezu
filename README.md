toriaezu
========

Install some tools and dotfiles.

How to use
----------

```console
git clone https://github.com/shakiyam/toriaezu
cd toriaezu
./provision.sh
```

Target OS
---------

* Oracle Linux Server 7.5 with UEK4
* Oracle Linux Server 6.10 with UEK4 (experimental)
* Ubuntu 16.04 LTS
